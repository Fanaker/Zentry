# Zentry

Modern personal finance platform designed to centralize expense tracking, income management, internal transfers, balances, and financial analytics in a single system.

---

## Features

* Expense and income tracking
* Internal transfer management (TI)
* Multi-account balance control
* Financial dashboards and analytics
* Excel data import
* Excel backup export
* Categorized financial movements
* Personal loan tracking
* Monthly financial summaries
* Responsive and modern UI

---

## Tech Stack

* Frontend: React + TypeScript
* Backend: Supabase
* Database: PostgreSQL
* UI: Tailwind CSS
* Deployment: Lovable

---

## Main Modules

### Financial Movements

Register and manage:

* Expenses
* Income
* Internal transfers

Each movement includes:

* Date
* Category
* Payment method
* Description
* Reference person
* Currency handling

---

### Dashboard

Visual financial overview including:

* Monthly expenses
* Monthly income
* Net balance
* Account balances
* Category distribution
* Financial trends

---

### Account Management

Track balances across:

* Debit cards
* Credit cards
* Cash
* Digital wallets

---

### Backup System

Manual Excel export system for secure local backups and data portability.

---

## Purpose

This project was created as a personal financial management system focused on organization, visibility, and long-term scalability beyond traditional spreadsheet-based workflows.
